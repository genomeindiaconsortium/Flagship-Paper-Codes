# GenomeIndia Analysis & Code RepositoryWelcome to the analytical codebase for the **GenomeIndia** project. This repository contains the complete suite of pipelines, downstream analysis scripts, and visualization code used to process, annotate, and analyze population-scale whole-genome sequencing (WGS) data. ---## Table of Contents[Module Overview]    * [1. Primary Processing & Variant Calling]    * [2. Variant Annotation & Clinical Impact]    * [3. Imputation Pipeline]    * [4. Pharmacogenomics (PGx)]    * [5. Plotting & Figure Generation]The repository is modularly organized into distinct analytical stages. Note that several sub-directories contain their own detailed README files or execution logs.GenomeIndia_codes/├── DRAGEN_pipeline/   # Variant processing & Joint calling       ├── Imputation/               # Phasing, imputation, and QC metrics related to imputation.├── pdMM_LoF_ACMG_Clinvar/    # Pathogenic, pDMM & Loss-of-Function variant analysis├── Pharmacogenomics/         # PGx analysis and star-allele calling├── Plotting_codes/           # R and Python scripts for Fig 1 - Fig 5├── Variant_annotation/       # VEP annotation for LoF variants including environment yaml file.Additional files:

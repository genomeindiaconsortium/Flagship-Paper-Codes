## This is a README for the R code to generate Figure 3C. A scatterplot of the average length of ROH (LROH) in centimorgans (cM) vs the number of ROH segments (NROH) of 83 GI populations. The color and shape of the GI populations are according to the ethnolinguistic group.

## Required Input file is Table S7.1. Summary of ROH measures across all the populations. SROH: Sum of ROH, NROH: Number of ROH, LROH: Average Length of ROH, FROH: the genomic proportion in ROH. p values are one-sided Kolmogorov–Smirnov tests with Benjamini–Hochberg FDR correction.

## Table S7.1 is available in the Supplementary Table, as specified in the Code.

The data used for generating this Sankey plot are embedded within the input file (`sankey_df.txt`) and the structure of the file is explained within the code. It corresponds to variants and associated phenotypes described in the supplementary tables S11.2  &  S11.3 of the manuscript. Some elements of the final composite figure were generated or arranged separately during manuscript figure preparation. Running this script produces the Sankey diagram in Fig4B illustrating variant consequences, gene associations, and medical relevance.

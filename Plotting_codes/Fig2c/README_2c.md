## Data Availability
The input data files should be placed in their respective directories (admixture_2c/) relative to the scripts. 

## Repository Structure & Code Descriptions

## ADMIXTURE Plot for 82 Populations (Figure 2c)
* Script Name: plot_admixture_fig2c.R
* Description: Generates Figure 2c, an ADMIXTURE plot visualizing the ancestry proportions (K=5) for 82 populations from the Genome India dataset. The populations are arranged and colored according to their admixture proportions, with annotations for Linguistic Group (LG) and Broad Group (BG).
* Input Data: k5_admix_data.csv (Expected in an admixture_2c/ directory)
